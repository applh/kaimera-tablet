# Features

## Camera Applet
The Camera applet has been enhanced with professional video recording capabilities.

### Key Features
- **Photo/Video Switch**: Toggle between Photo and Video modes using the selector below the viewfinder.
- **Video Recording**:
    - Tap the Red button to start recording.
    - Timer displays recording duration.
    - Tap again to stop. Videos are saved to `Movies/Kaimera`.
- **Zoom**: Pinch to zoom from 1x up to maximum digital zoom.

## Files Applet
The Files applet manages your captured media.

### Key Features
- **Unified Gallery**: View both Photos and Videos in a single grid.
- **Video Identification**: Videos are marked with a "Play" icon overlay.
- **In-App Media Viewer**:
    - Tap any file to view it in full screen.
    - **Photos**: Supports pinch-to-zoom and pan.
    - **Videos**: Includes playback controls (Play/Pause/Seek) and zoom.
- **Management (CRUD)**:
    - **Long Press** any file to open the context menu.
    - **Rename**: Change the filename.
    - **Delete**: Remove the file permanently (requires system permission on Android 10+).

## Downloads Applet
The Downloads applet manages files downloaded from the web.

### Key Features
- **Downloads List**: View and manage all downloaded files.
- **Manual Download**: Add new download URLs directly by clicking the **+** button.
- **Storage Tracking**: Monitor available device storage space.
- **Quick Actions**: Open or delete files directly from the list.
