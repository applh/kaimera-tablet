# Kaimera Tablet - User Guide

## Introduction
Kaimera Tablet is a productivity and media application designed for efficiency. This guide covers the main features available in version 0.0.8.

## Camera Applet

### Photo Mode
- **Capture**: Tap the large white shutter button to take a photo. Photos are saved to your device's Gallery.
- **Zoom**: Use the vertical slider on the right to zoom in/out (1x - Max).
- **Focus**: Tap anywhere on the preview to focus and adjust exposure. A yellow circle helps verify the focus point.
- **Flash**: Cycle through Auto/On/Off using the lightning icon in the top right.
- **Camera Switch**: Toggle between Front and Back cameras.

### Video Mode
- **Switching**: Tap the video camera icon in the "Compact Mode Switch" (pill shape).
- **Recording**: Tap the red shutter button to start recording. A duration timer will appear. Tap again to stop.
- **Storage**: Videos are saved to the "Movies/Kaimera" folder.

### Settings
Access settings via the gear icon.
- **Grid Lines**: Overlay a grid (0-10 rows/cols) to help with composition.
- **Timer**: Set a delay (0-100s) before capture.
- **Flash Mode**: Set default flash behavior.

## Files Applet
- Browse local files.
- **Note**: Requires "Manage All Files" permission on Android 11+.

## Troubleshooting
- **Permission Denied**: If the camera or microphone doesn't work, ensure you have granted permissions in Android Settings > Apps > Kaimera Tablet > Permissions.
