# User Installation Guide

## Getting Started

1.  **Prerequisites**: You need an Android Tablet (Xiaomi Pad 2 Pro or compatible) and a computer with USB.
2.  **Get the APK**: Download `app-debug.apk` from the releases or build it yourself.

## Installation via ADB

1.  Enable **Developer Options** and **USB Debugging** on your tablet.
2.  Connect tablet via USB.
3.  Run:
    ```bash
    adb install -r app-debug.apk
    ```

## Usage

- **Launcher**: The main screen shows available applets.
- **Camera**: Tap the Camera icon to open.
- **Settings**: Tap the Settings icon to configure options.
